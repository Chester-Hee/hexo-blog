# API压测标准

## 东风日产

> 服务器情况：20台8核16G
> 压测时间：6分钟

|选项|值|
|---|---|
|并发|2600|
|TPS|接近900|

换算为单台机器的性能

|选项|值|
|---|---|
|并发|130|
|TPS|45|

因为集群是会有性能损耗的，所以需要把损耗算上，按15%来计算

> 总并发量 = 单机并发量 x 机器数量 x (1-15%)
> 总TPS = 单机TPS x 机器数量 x (1-15%)

最终单机性能如下：

|选项|值|
|---|---|
|并发|150|
|TPS|50|
