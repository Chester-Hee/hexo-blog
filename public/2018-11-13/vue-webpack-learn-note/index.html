Vue 安装

  • CDN

这个一个很好的开始，直接在页面引入 script 标签，就可以开启 Vue 的世界！

当然如果你担心直接引入 script 标签会有一天失效，可以下载文件保存到自己的服务器中，然后引用自己的服务链接。我开发工作中也经常这么干，缺点就是如果访问流量较大，还是需要加 CDN 进行缓冲。

  • Webpack
